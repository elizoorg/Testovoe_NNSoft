Тестовое для компании 
Что реализовано:
- Автоматически обновляемый список запущенных процессов(обновляется раз в 10 секунд)
- POST запросы при нажатии на кнопки Get data и Send data
(от сервера возвращается пустая строка)
-  Строки шифруются через XOR + base64
-  Каждая строка уникальна т.к. к ней в начале добавляется уникальные идентификатор из 8 символов
-  RID генерируется уникально при включении приложения
-  Запрет на запуск двух экземпляров
-  Запуск с правами родительского процесса + возможность повышения уровня привелегий при нажатии на соответствующую кнопку
-  Изменение интерфейса при изменение размеров окна
  Приложение проверенно на чистых Windows 10,11  без установленных дополнительных пакетов
- Закрытие приложения вызывает сворачивание в трей а не закрытие
- Реализовано (?) меню в трее. Код вызывает ошибку прав доступа (код ошибки 0x80070015) что не дает полноценно протестировать код для работы с треем.
